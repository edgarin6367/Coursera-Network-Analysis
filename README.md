# Network Traffic Analysis & Decryption Lab

This repositorie has scripts and docs about Network Traffic capture using `TCPDump` and the decrypted traffic TLS by `SSLKEYLOGFILE`.

# Tools
- **OS:** Arch Linux
- **Capture:** tcpdump
- **Analysis:** Wireshark
- **Documentation:** Obsidian & Excalidraw

## Projects
### 1. Checkspy (SSH Monitoring)
Monitoring script for port 22, look for logs and keep persistence to interrupted data (`Ctrl + C`).
### 2. TLS Decryption Diagram
>[!NOTE]
> This project form part of my learning at Coursera about TCPDump and Traffic Data Analysis.
